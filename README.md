# Quizzical

> A simple React quiz app built with Vite.

## About

Quizzical is a lightweight, interactive quiz application built with React. It presents questions to users, collects answers, and provides feedback – perfect for learning or quick trivia sessions.

## Features

- Dynamic question rendering
- Multiple-choice answer selection
- Score tracking at the end of the quiz
- Responsive design (powered by custom CSS)

## Tech Stack

- **React** – UI library
- **Vite** – Build tool and development server
- **JavaScript (ES6+)** – Core logic
- **CSS** – Styling (65% of the codebase)
- **HTML** – Base structure

## Getting Started

Follow these instructions to run the project locally.

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or later)
- npm (comes with Node.js)

### Installation

```bash
# Clone the repository
git clone https://github.com/pyticph/quizzical.git

# Navigate into the project folder
cd quizzical

# Install dependencies
npm install

```
bash
npm run dev
```
Open http://localhost:5173 to view it in your browser.
